Часть1. Описание приложения

1. Верхная панель

1.1. Выборочное меню "Файл"

При нажатие на кнопку "Файл" появляется меню, состоящееся из кнопок:
- "Создать", которая создает новый проект;
- "Сохранить", которая сохраняет текущий проект;
- "Загрузить", которая загружает готовый проект;
- "Выйти", которая закрывает приложение.

1.2. Кнопка "Сущность"

Эта кнопка создает новую сущность на карте. Для создания сущности надо нажимать на кнопку и кликнуть в нужное место на карте.

1.3. Кнопка "Связь"

Эта кнопка создает связь между двумя уже существующими сущностями. Для создания связи надо нажимать на кнопку и далее на сущности, которые нужно связывать.

1.4. Кнопка "Выделить"

Эта кнопка помогает выделить нужный объект (сущность, связь) для его редактирования. Для выделения объекта надо нажимать на кнопку и далее на нужный объект.

2.4. Кнопка "Сохранить"

Она сохраняет изменения после редактирования объекта.

2.5. Кнопка "Удалить"

Она удаляет объект в режиме редактирования.

2.6. Кнопка "Добавить поле"

Эта кнопка появляется только в режиме редактирования сущности (после выделения). Она позволяет добавить новое поле в сущность.

3. Вкладка "Работа"

3.1. Пункт "Получатель"

Этот пункт состоит из выборочного меню. В этом меню можно выбрать сущность, которая будет получать посылку.

3.2. Пункт "Отправитель"

Этот пункт состоит из выборочного меню. В этом меню можно выбрат сущность, которая будет отправлять посылку.

3.3. Кнопка "Отправить"

Эта кнопка запускает вычисление оптимального пути отправки посылки.

Часть2. Инструкции по использованию

1. Редактирование сущности

Прежде всего нужно выделить данную сущность. Тогда в правой части окна под кнопками появляются текущие параметры сущности. Первый пункт - это название сущности. Его можно изменить просто скорректируя поле, в котором оно написано. Второй пункт - это тип сущности. Его можно изменить вибирая нужный тип из предоставленного меню. Дальше идут сведения о скорости и радиусе действия, которые не возможно изменить. Можно добавить поле нажатием на соответсвующую кнопку. Его также можно редактировать : выбрать тип, изменить название и значение. После внесения изменений нужно сохранить новое состояние сущности нажатием на соответсвующую кнопку. И удалить сущности тоже возможно нажатием на соответсвующую кнопку.

2. Редактирование связи

Прежде всего нужно выделить данную связь. Тогда в правой части окна под кнопками появляются текущие параметры связи. Первый пункт - это название связи. Его можно изменить просто скорректируя поле, в котором оно написано. Второй пункт - это ключевое слово. Его можно изменить просто скорректируя поле, в котором оно написано. Дальше идут сведения о типе связи, которые можно изменить, отмечая нужные поле.

3. Поиск оптимального пути

Сначала необходимо переходить в вкладку "Работа". Первый пункт - это получатель. Его нужно задать вибирая нужную сущность из предоставленного списка. Второй пункт - это отправитель. Его нужно задать вибирая нужную сущность из предоставленного списка. Затем надо нажимать на кнопку "Отправить" и оптимальный путь вычисляется.
