Документация пользователя:
Инструкция по сборке:
1)С сайта QT:
 http://qt-project.org/downloads
Скачиваем Qt 5.2.1 for Windows (MinGW 4.8, OpenGL)
2) Открываем Qt creater, при необходимости настраиваем его:
В пункте "Проекты" изменяем каталог сборки для проекта. 
3) Открываем проект. (Путь к проекту не должен содержать символов из кириллицы)
4) В Qt Creater жмем большую зелёную стрелку, и проект компилируется + запускается.
5) При необходимости запуска exe-файла, в папку с файлом следует поместить дополнительные dll-файлы.
При запуске будет сообщено какие файлы необходимы. Их можно найти Командой поиск в директории среды QT.
Для сборки релизной версии необходимо действовать так:
http://qt-project.org/wiki/Deploy_an_Application_on_Windows
Создано 3 проекта: 
1) для отлаживания контейнера: Contaner располагается в папке: scr/Contaner
2) проект с реализацией интерфейса и контейнера: /interface/DemoInterfaces
3) сериализация: /serialization
